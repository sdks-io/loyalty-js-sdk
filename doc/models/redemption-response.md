
# Redemption Response

*This model accepts additional fields of type unknown.*

## Structure

`RedemptionResponse`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `referenceId` | `string` | Required | A unique Reference ID for the transaction, generated by the partner.<br><br>**Constraints**: *Maximum Length*: `128` |
| `transactionDateTime` | `string` | Required | The time and date of the redemption transaction. |
| `updatedPointBalance` | `string` | Required | The user's current loyalty point balance after the requested items have been redeemed. |
| `redemptionReqArray` | [`RedemptionResponseItem[]`](../../doc/models/redemption-response-item.md) | Required | This array requests the points to be redeemed from the users point balance for the items requested. The same information will be shown to the user. Multiple items can be redeemed in a single API request. |
| `additionalProperties` | `Record<string, unknown>` | Optional | - |
